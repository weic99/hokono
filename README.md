# Hokono

> A place for pets
  > - Explore and follow pets from around the world
  > - Document your own pet's life
  > - Adopt from nearby shelters

## Usage

Run ```npm install``` to install all node modules.

Run ```npm start``` to start the app.

## Requirements

- Node 0.10.x
- NPM
